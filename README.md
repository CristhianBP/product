# TESLO SHOP

## Pasos

1. clonar proyecto

```
 git clone url nameFolder
```

2. instalar paquetes de node modules

```
 npm install
```

3. clonar el archivo `.env.template` y renombrarlo a `.env`
4. cambiar las variables de entorno
5. levantar la aplicacion en modo dev

```
 npm run start:dev
```
